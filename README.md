# Server para Scraper
Se realizan 2 formas de scraper con 2 framework muy interesantes:
- Selenium
- Playwright
La imgen docker no ha sido para nada optimizada, al contrario se utiliza la recomendación de Playwright, eso lleva a una imagen de más de 2 Gb (lo cual es un brutalidad) pero se va a optimizar con el tiempo.
